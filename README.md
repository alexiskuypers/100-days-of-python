# 100 Days of Python

A collection of Python exercises and small projects completed as part of the **100 Days of Code: The Complete Python Pro Bootcamp** by Angela Yu.

## About This Repository

This repository contains the projects and exercises I complete throughout the course.

The goal is to practice Python regularly, strengthen my understanding of the language, and improve my ability to solve programming problems independently.

Each course day is stored in a separate directory with its related Python files.

## Objectives

Through this challenge, I aim to:

* practice Python consistently;
* reinforce Python fundamentals;
* improve my problem-solving skills;
* develop stronger coding habits;
* become more comfortable debugging programs;
* improve code readability and naming;
* gradually build more complex projects.

## Repository Structure

Each course day has its own directory.

```text
100-days-of-python/
├── day-001-band-name-generator/
│   └── band_name_generator.py
├── day-002-tip-calculator/
│   └── tip_calculator.py
├── day-003-project-name/
│   └── project_name.py
├── .gitignore
└── README.md
```

Directory naming convention:

```text
day-XXX-project-name/
```

Python files use lowercase `snake_case` names:

```text
project_name.py
```

## Running a Project

Clone the repository:

```bash
git clone https://github.com/alexiskuypers/100-days-of-python.git
cd 100-days-of-python
```

Run a project with Python:

```bash
python day-002-tip-calculator/tip_calculator.py
```

Most early projects only require Python and use the standard library.

Any additional dependencies will be documented when needed.

## Learning Approach

For each project, I try to:

1. understand the problem and expected result;
2. break the problem into smaller steps;
3. write my own solution;
4. test and debug the program;
5. review the readability of the code;
6. compare my approach with the course solution;
7. commit the completed project.
